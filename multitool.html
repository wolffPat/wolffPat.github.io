Features

This is a tool to help with indentifying potential phishing threats for user of office 365 ATP and use safelink encoding

It contains a search bar that functions as a safelinks decoder, url virus scanner, email auto generation, and much more

It also contains one click copy and paste responses for phishing emails.

It also contains the ability to hash malicous files and submit the hashvalue to virus total!
